# Wazuh SIEM

## Overview

This project demonstrates a basic SOC (Security Operations Center) lab using Wazuh SIEM for log monitoring, threat detection, and security event analysis.

## Features

* SSH brute-force attack detection
* Log analysis
* Security alert monitoring
* Incident investigation

## Project Workflow

* Kali Linux clone used as attacker machine
* Main kali linux used as target system
* Wazuh used to monitor logs and generate alerts

## Skills Demonstrated

* Linux administration
* SIEM monitoring
* Threat detection
* Log analysis
* SOC investigation basics
